Threshold Run - August 5

W10D3 – Threshold Run
Pace-based threshold stimulation
Total duration
46 minutes
Main work
4 x 5min threshold
Objective
Stimulate threshold without exhausting

Workout Structure

Warm-up (10 min)
6:30-6:45/km Very easy pace
4 repetitions of:
5 min at 5:00-5:10/km Threshold pace
1:30 at ~6:30/km Recovery jog
Cool-down (10 min)
6:45/km Easy jog

Key Points

  • Threshold pace is not all-out: Sustained but controlled, uncomfortable but not gasping
  • Cool weather advantage: You'll be able to maintain threshold pace without HR spiking too much
  • Garmin setup: Configure pace-based segments if clearer than HR zones
  • Don't finish exhausted: If feeling tight, ease off slightly in 3rd and 4th repetitions
